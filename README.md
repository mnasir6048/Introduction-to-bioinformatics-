 Brief Overview of Introduction to Bioinformatics
Bioinformatics is a field that combines biology, computer science, mathematics, and statistics to analyze biological data such as DNA, RNA, and protein sequences. It helps researchers understand genetic information, study diseases, and develop new medical treatments. Bioinformatics is widely used in genomics, proteomics, and computational biology.

 Description of the Assignment
This assignment was given to Group 5 to introduce students to the basic concepts of bioinformatics and the use of collaborative tools in research.
